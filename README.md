# Temperature Converter

A simple Python program that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Features
- Celsius to Fahrenheit & Kelvin
- Fahrenheit to Celsius & Kelvin
- Kelvin to Celsius & Fahrenheit

## How to Run

1. Make sure Python is installed.
2. Run the program:

```bash
python temperature_converter.py
```

3. Enter the temperature scale and value.
4. The program will display the converted temperatures.

## Example

Input:
```
Scale: Celsius
Temperature: 25
```

Output:
```
Celsius: 25.00 °C
Fahrenheit: 77.00 °F
Kelvin: 298.15 K
```
